Display - Block e Inline

** CAIXAS BLOCK:

        
            **Block**
            - Ocupa toda a linha e colocar o proximo elemento abaixo desse.
            -- width e heigth são respeitados.
            -- padding, margin, border irão funinar normalmente.
            

            Exemplos de BLOCK por PADRAO:
            <p> <div> <section> e todos os <H1> <H2> ...
BLOCK -> Ocupa toda linha colocando o proximo elemento abaixo desse (CAIXA 1)
BLOCK -> Ocupa toda linha colocando o proximo elemento abaixo desse (CAIXA 2)

** CAIXAS INLINE:

        
            **Inline**
            - Elementos ao lado do outro.
            -- width e heidth NÃO FUNCIONA.
            -- Valores horizontais de margin e padding funciona.
            

            Exemplos de BLOCK por PADRAO:
            <a> <strong> <span> e <em>
INLINE -> colocar as caixas na mesma linha ao lado (CAIXA 1) INLINE -> colocar as caixas na mesma linha ao lado (CAIXA 2)
