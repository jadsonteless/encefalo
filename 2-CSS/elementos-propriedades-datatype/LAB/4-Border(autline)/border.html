Border & Outline

** Propriedades do Border:

  • <border-styler>
    • solid
    • dotted
    • dashed
    • groove
    • double
    • ridge
    • inset
    • outset
  • <border-width>
    • <length>
    • <porcente>
  • <border-color>
    • adicionar cor

** Exemplod de alguns codigos e suas funcionalidades.

            
                div {
                    border-botton: none (Apagar a barra escolhida ex: botton)
                    border-botton: cor (pinta a barra escolhida ex: botton)
                    border-botton: groove (troca o stylo da barra escolhida ex: botton)
                }
