Box-sizing

Borda de 100px

Exemplo 01: Mesmo com a configuração de 100px na caixa, apos adicionar conteudo os valores de tamanho e altura da caixa vai alterando se o uso do box-sizing

            
                .exp1 {
                    width: 100px;
                    height: 100px;
                    border: 1px solid red;
                    margin: 5%;
                    padding: 0px 20px;
                }
Borda de 100px

Exemplo 02: Utilizando box-sizing o tamanho da caixa se permanese fixa idependente do conteudo dentro.

            
                .exp2 {
                    width: 100px;
                    height: 100px;
                    border: 1px solid red;
                    margin: 5%;
                    padding: 0px 20px;
                    box-sizing: border-box;
                }
